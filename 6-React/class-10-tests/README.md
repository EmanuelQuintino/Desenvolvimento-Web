# Class 10 Automatic Tests

## Vitest

```bash
# install vitest for tests
npm install vitest -D
```

## Testing Library

```bash
# install testing library
npm install @testing-library/react -D
```

## JSDOM

```bash
# install jsdom for render dom
npm install jsdom -D
```
