# Class Markdown (md)

## Titles

Use # to create titles. Add more # for sub-headings, indicating lower levels:

```md
# Title Level 1

## Title Level 2

### Title Level 3

#### Title Level 4

##### Title Level 5

###### Title Level 6
```

## Emphasis

```md
highlight: `text`.
bold: **text**.
italics: _text_.
strikethrough: ~~text~~.
```

## Lists

- Unorder lists: Use -, \*, or + followed by a space for list items.

```md
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2
```

- Order lists: Use numbers followed by a period and space.

```md
1. First Item
2. Second Item
   1. Subitem
   2. Subitem
```

## Links

```md
[Google](https://www.google.com)
```

## Images

```md
![OpenAI Logo](https://example.com/logo.png)
```

## Code blocks

````md
```js
console.log();
```
````

## Quotes

```md
> This is a quote
```

## Table

```md
| Heading 1 | Heading 2 |
| --------- | --------- |
| Data 1    | Data 2    |
| Data 3    | Data 4    |
```
